## Scope

This repository presents:
- a general constitutional and governance proposal for AI systems under finite capacity
- companion instruments for force, applicability, projection, evidence, interpretive limits, and policy translation
- a public-safe reference implementation surface showing how minimum reviewable artefacts may be generated

This repository does **not** contain the full TER runtime.

## Reading guidance
