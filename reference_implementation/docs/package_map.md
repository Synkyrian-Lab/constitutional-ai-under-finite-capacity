# Package Map

## Primary purpose
This repository demonstrates a **minimum reviewable evidence surface** for Constitutional AI under finite capacity.

## Public artefacts
- `manifest.json` — package identity and scope
- `actions.jsonl` — explicit constitutional acts
- `certificate.json` — bounded verdict surface
- `checksums.sha256` — integrity layer

## Intended interpretation
This repository should be read as:
- **implementable**
- **reviewable**
- **public-safe**
- **non-exhaustive**

It should not be read as:
- the only valid implementation
- a legal compliance tool
- the full TER runtime
- a complete audit framework
